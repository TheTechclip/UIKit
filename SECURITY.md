# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within @musecat/uikit, please report it by emailing the maintainers at **heart@musecat.app**.

Do NOT disclose the vulnerability publicly until it has been addressed.

We will acknowledge receipt within 48 hours and work to resolve the issue promptly.

## Scope

This security policy applies to all packages under the `@musecat` organization on npm.
