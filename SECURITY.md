# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within @musecat/uikit, report it privately at **heart@musecat.app**. Include the affected package/version, reproduction steps, impact, and—if available—a minimal proof of concept.

Do NOT disclose the vulnerability publicly until it has been addressed.

Do not disclose the vulnerability publicly until the maintainers confirm that a fix or mitigation is available. We aim to acknowledge reports within 48 hours and will provide status updates while triaging a valid report.

## Scope

This policy covers the published `@musecat/uikit` package, this repository's source code, and its documented Next.js integration. It does not cover vulnerabilities exclusively in third-party dependencies; please report those to the upstream project as well.
